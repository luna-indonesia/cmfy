### NodeJS

You  have to at least have `NodeJS version 16++` installed.

please check https://nodejs.org/en/download/ for further information.

## Steps

### Download CMFY

We're using `degit` by Rich Harris, it's a good straight forward project scaffolding.

In your `terminal` / `command prompt`, execute `npx degit`

Your system will download CMFY and install it to thed designate folder/directory.

### Install Dependencies

In your `terminal` / `command prompt`, execute `npm i`

### Run The Server

In your `terminal` / `command prompt`, execute `npm run dev`
